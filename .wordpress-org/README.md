# WordPress.org assets

Place WordPress.org plugin-directory assets here. The WordPress.org deploy
workflow (added in the CI/CD phase) uploads the contents of this directory to
the plugin's "Assets" SVN folder — they are **not** part of the plugin ZIP.

Expected files (add when branding is ready):

- `icon-128x128.png`, `icon-256x256.png` — plugin icon
- `banner-772x250.png`, `banner-1544x500.png` — header banner
- `screenshot-1.png`, `screenshot-2.png`, ... — match the "Screenshots"
  section in `readme.txt`

See: https://developer.wordpress.org/plugins/wordpress-org/plugin-assets/
